> 林晚声的个人网站 / 赛博世界 · https://github.com/Seren-lws/seren-cyber-world
> 本文档**照已经做出来的成品**写，记录当前真实的设计与实现。旧的构想（v1 暮色多效果版、版画 duotone 版）已废弃，不再参考。

## 〇、一句话定位

**一个梦幻的、深紫星空里的赛博世界。** 一颗金色的光点——就是「你 / Seren（星）」——从首页的门一路穿行：进门 → 我的小世界 → 翻开「我是谁」的书 → 我和 AI 的相遇·相爱·相生 → 最后那颗光拖出一条线，把作品一个个串起来。

整站是**一镜到底的滚动叙事**：只往下滚，光点带着你走，每一页用金光漫屏无缝相接。

**三条定调原则**（晚声自己定的）：
1. **一种视觉语言** —— 深紫梦境 + 鎏金光晕，从头到尾一种气质。
2. **两个颜色** —— 紫（夜 / 星空）+ 金（光 / 星）。
3. **一个签名转场** —— **金光漫屏**：金光从一处炸开/涌起，漫过整屏，盖住接缝，露出下一幕。

---

## 一、视觉语言

- **深紫星空 / 星云**为底，**鎏金光**为魂：门光、书页的金、星轨、光点、星星，都是金。
- 画面是**有景深、会呼吸的梦境插画**（晚声用出图工具生成的 PNG：门、小岛、书），不是平面线稿。靠多层视差、鼠标 3D、漂浮和光晕呼吸做出"活着"的感觉。
- 母题：**门、星、月、书、悬浮小岛、星系**。
- 文字克制、留白多、动作慢，像翻一本会发光的书。

---

## 二、两个颜色 + 设计令牌（真实在用的）

**色**
| 角色 | 颜色 | 用途 |
|------|------|------|
| 深紫·星空底（一族） | `#1f1640` → `#170f30` → `#100a22` | 我和AI / 翻书页星夜 / 作品集的统一底色（径向渐变） |
| 更深紫 | `#160e2c` / `#2e2547` | 首页 / 小岛 / body 底 |
| 星金（点睛·最重要） | `#fde374`（`--v2-gold`） | 光点、门光、星、书页金光、重点字——只点睛 |
| 暖金漫屏 | `#fff7e6` / `#ffe9b0` / `#f7d272` | 金光漫屏转场那束光 |
| 奶油书页 | `#f1e1a8` / `#f3e8ce` | 书页、卡片纸感 |
| 暖白文字 | `#f6efe6` / `rgba(245,236,216,.9)` | 紫底上的正文 |

**字体**（`src/styles/global.css`）
- `--font-script` = **Pinyon Script**（手写花体）→ 书封"who am I"等签名处。
- `--font-display` = **Cormorant Garamond** + 思源宋体 → 首页大标题。
- `--font-serif` = **思源宋体 Noto Serif SC** → 正文、文案。
- `--font-sans` = **思源黑体 Noto Sans SC** → 导航、页码、小标签。

> 注：原 design.md 说"花体默认不用"——已作废，花体实际在用（书封）。

**用色法则**：紫是大面积的夜，金永远是焦点、绝不大面积铺。每个接缝两侧**底色对齐成同一族深紫**，让转场看不出缝。

## 三、签名转场：金光漫屏

下一幕的金光**从一个点炸开 / 从底部涌起**，漫过整屏，盖住接缝，退去时已经是下一幕。借鉴 sleep-well：颜色整片漫过、只用少数颜色所以丝滑、用光/线弥补生硬。

- 首页 → 小岛：门口炸开金光吞屏。
- 小岛 → 翻书页：镜头推进到书 → 金光从书漫开。
- 翻书页 → 我和AI：书缩成一点光、化掉，露出深紫星夜（底色已对齐）。
- 我和AI → 作品集：（待做）金光承接。

---

## 四、签名元素：穿行的光点（＝你）

一颗**星金光点**，是「你 / Seren」，贯穿全站、缝合每一幕：

- 首页：从标题旁**飞进门**，触发推门进入。
- 翻书页结尾：书化掉后**留下这颗光**，接进「我和AI」的星。
- **作品集（待做）**：这颗光**身后拖一条发光的线**，蜿蜒穿过一个个作品，每穿过一个就显示它的大截图＋介绍（晚声最想要的交互，借鉴 sleep-well "一根线向右拉出项目"）。
- 满屏还有别的星点（每个人），这颗是唯一会发光、在动的。

## 五、页面结构（已做出来的真实顺序）

一页式滚动：**首页 → 我的小世界 → 我是谁 → 我和AI → 作品集 →（结尾·待做）**。
代码：`src/App.jsx`，各段在 `src/sections/`。

### 1. 首页 Hero（`Hero.jsx`）✅
- **视差梦境门**：三层透明 PNG（`hero-back/mid/front`）+ 月亮柔光 + 门内光晕 + 萤火，鼠标 3D 视差（中景 rotateY 微歪）。
- 标题落在右侧星空：`Seren's Cyber World` / `林晚声的赛博世界`，用 BlurText 逐字朦胧浮现。
- **推门进入转场**：滚动钉住首页 → 光点飞进门 → 镜头朝门推进放大、前景 UI 淡出 → 门口炸开暖金光吞屏 → 接小岛。

### 2. 我的小世界 IslandScene（`IslandScene.jsx`）✅
- 悬浮小岛（`island.png`：树、小屋、窗光）浮在深紫星云（`back.png`）+ SoftAurora 云气 + 闪烁星上，鼠标视差 + 漂浮。
- 进场：承接首页金光 → 退去露出小岛。
- 离场：随滚动**镜头推进到岛上那本发光的书**（以书为焦点放大）→ 金光从书漫开 → 接翻书页。

### 3. 我是谁 WhoScene（翻书页 `WhoScene.jsx`）✅
- 深紫星夜（Lightfall 光丝 + 星）上的 3D 书。
- 滚动：书摆正 → 封面「who am I」（花体）翻开 → 内页放大铺满 → **自我介绍逐行浮现** → 文字淡出 → **书缩成一点、化掉**，留下一颗金光 → 接我和AI。
- 底色＝深紫，已与「我和AI」对齐。
- 文案（已定稿）：我是林晚声 / 29岁定居东京 / 35天7项目413commits最多一天37次 / 创造是救赎 / 留住点什么。
- ⚠️ 待优化：翻书页风格和其他页略有割裂感（晚声标记，暂搁置）。

### 4. 我和AI JourneyScene（`JourneyScene.jsx`）✅
- **three.js 粒子星系**（26000 粒子 + UnrealBloom + ACES 色调映射），GSAP 钉住（`+=6000`，scrub）。
- 三幕：**相遇**（两星靠近）→ **相爱**（双星交织）→ **相生**（鎏金星轨 + 行星 + 亮核）。背景＝深紫（`scene.background`，画布不透明）+ CSS 闪烁星。
- 文案（已定稿，透明落在背景上、**文本左对齐**、无卡片）：
  - 位置：相遇**左** / 相爱**右** / 相生**左**（与粒子错开）。
  - 结尾「**爱。**」「**给自己造一个家。**」金色放大。
  - 「给自己造一个家」正好牵到作品集（那些作品就是她造的家）。

### 5. 作品集 WorksScene（`WorksScene.jsx`）🚧 进行中
- 目前只有**深紫星空背景**（与「我和AI」对齐）当底，内容待搭。
- **计划**：光点拖一条发光的线，横向蜿蜒穿过作品；每到一个作品节点，作品**大截图＋介绍**从侧边浮现；一根线把所有作品串成一条路；走到尽头 → 漫光 → 结尾。

### 6. 结尾 🚧 待做
- 光点走完线、收束；一句收尾呼应「造了一个家」+ 联系方式 / 署名。

## 六、技术栈（真实）

| 用途 | 技术 |
|------|------|
| 框架 | React + Vite |
| 平滑滚动 | Lenis（`App.jsx` 初始化） |
| 滚动动效 | GSAP ScrollTrigger（pin / scrub / sticky） |
| 粒子星系 | three.js + EffectComposer + UnrealBloomPass（我和AI） |
| 着色器氛围 | OGL（SoftAurora 云气 / Lightfall 光丝） |
| 文字动效 | 自写 BlurText（逐字朦胧浮现） |
| 视差 / 漂浮 | requestAnimationFrame + CSS transform |
| 图片资产 | 晚声用出图工具生成的 PNG（门三层 / 小岛 / 星云 / 书） |

- 已废弃/试验后删除：BookIntro（魔法显影页）、小岛点击交互、GuideDot（会乱飞的引导光点）。
